# Two Pointer Pattern

## Core Intuition
Use two indices to reduce nested loops into linear scan.

## When To Use
- Sorted array
- Pair sum problems
- Removing duplicates
- Fast/slow pointer scenarios

## Variations
- Opposite ends (left/right)
- Same direction (slow/fast)
- Partitioning

## Time Complexity
Usually O(n)

## Common Mistake
Forgetting sorted constraint.
